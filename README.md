
# Transformerless Power Supply

A transformerless AC to DC power supply designed using KiCad.

## Features
- 230V AC input
- 5V DC output
- Compact PCB design

## Software Used
- KiCad

## Files Included
- Schematic
- PCB layout
- Gerber files
- PCB images

⚠ WARNING:
This project directly interfaces with AC mains voltage.
Handle carefully.
